## Requirements

* Ruby 3.2.2
* Node.js & npm
* SQLite for local
* PostgreSQL for deploy

## Setup

```bash
make setup
```

## How to run the test suite

To run tests:
```bash
make test
```
To run linters:
```bash
make lint
```
Or both:
```bash
make check
```
